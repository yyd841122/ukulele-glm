## ⏭️ Phase 1：MVP — 听得准 + 能弹唱一首歌

**核心目标**：验证「AI 实时听音评分」技术闭环，让用户能用 App 完成「调音 → 选谱 → 跟弹评分 → 打卡」完整链路。

### 范围

| 模块 | MVP 范围 |
|------|---------|
| 账号 | 手机号登录、基础资料 |
| 音频引擎 | **YIN 实时识别**（先 Dart 插件，不达标下沉原生）、Onset 节奏识别、输入校准 |
| 工具 | **智能调音器**、**节拍器**、**和弦库**（C/Am/F/G/Em/Dm 等基础和弦） |
| 曲谱 | 曲谱库（30+ 首入门弹唱谱）、滚动播放器、变速/变调 |
| 跟弹 | 曲谱**跟弹模式**（AI 实时评分音准+节奏）— 核心验证 |
| 我的 | 练习数据看板（时长/曲目数）、每日打卡 |
| 全局 | 音频权限引导、麦克风输入校准 |

### 关键里程碑

| # | 里程碑 | 验收标准 | 预估 |
|---|--------|---------|------|
| M1 | **PoC：乐音识别验证** | Flutter 端识别尤克里里 4 空弦正确率 >95%，延迟 <150ms | 1.5 周 |
| M2 | 工具集可用 | 调音器/节拍器/和弦库三大工具上线 | 1.5 周 |
| M3 | 曲谱库 + 播放器 | 30 首曲谱可浏览、滚动播放、变速变调 | 2 周 |
| M4 | **跟弹评分闭环** | 跟弹模式实时反馈 ✓/✗ + 评分报告 | 2 周 |
| M5 | 打卡 + 账号 + 打包发布 | 内测包可装机使用 | 1 周 |

### 交付标准（Definition of Done）
- 新用户可在 10 分钟内完成：登录 → 调音 → 选谱 → 跟弹一首歌 → 看到评分 → 打卡；
- 音高识别准确率 >90%（标准环境）；
- 崩溃率 <0.5%（MVP 放宽）。

## Phase 2：V1 完善 — 系统学习 + 游戏化留存

**核心目标**：从「能弹一首」到「系统学起来、愿意留下来」。

### 范围
- **互动视频教学**：教学视频分段播放 + 老师示范 → 用户跟弹 → AI 评测；
- **必修课课程体系**：30 节体系化课程（持琴→调音→和弦→扫弦→第一首弹唱→指弹入门）；
- **节奏/扫弦练习**、**和弦转换练习**；
- **游戏化**：连续打卡、成就/勋章、经验等级；
- **学习目标** + 基于规则的个性化推荐；
- **数据云同步**（账号体系打通后端）。

### 练习模式分级（渐进式，对标 AI 音乐学园）
| 模式 | 说明 | 状态 |
|------|------|------|
| **逐音等待模式** | 新手用：弹对才走，无限等待，一个音一个音练 | ✅ 已实现 |
| **整曲跟随模式** | 熟练用：按正常节奏不停走，漏弹/错弹只标记不中断，弹完整首出总分（命中率+音准+节奏） | 📋 待实现（V1） |

> 整曲跟随模式是 AI 音乐学园的核心体验——还原真实弹唱场景。
> 依赖更精准的实时识别（流动判定，不能等用户弹准），待识别稳定后实现。

### 关键里程碑
| 里程碑 | 说明 |
|--------|------|
| 互动视频教学引擎 | 分段播放 + Call&Response + 评分 |
| 必修课内容生产 | 30 节课脚本与录制 |
| 游戏化系统 | 打卡/成就/等级上线 |
| 后端 + 云同步 | 用户/进度/成就数据上云 |

### 交付标准
- 课程体系可连贯学习，互动教学评分可用；
- 次日留存 >35%，7 日留存 >25%。

## Phase 3：V2 增长 — 内容生态 + 商业化

**核心目标**：内容规模化、社交化、开始赚钱。

### 范围
- **选修课 / Pro 课 / 风格课程**（夏威夷风、指弹独奏、技巧专项）；
- **曲谱 UGC**：用户上传曲谱、审核上架；
- **曲谱导入**：支持 MusicXML / ChordPro；
- **排行榜**（周/月练习榜）、**好友 PK**；
- **会员体系上线**：永久会员 / 月年订阅（见 PRD §8）；
- **离线下载**、推送通知完善；
- **高精度评测**（云端 CREPE/FCPE 兜底，复杂弹唱评分）。

### 交付标准
- 曲谱库突破 500+，UGC 流程跑通；
- 会员转化率达标（待定），ARPU 健康。

---

## Phase 4：远期 — 多乐器 + 多端 + AI 智能私教

**核心目标**：平台化扩展。

### 范围
- **横向多乐器**：吉他、钢琴、声乐等（架构已预留多乐器配置）；
- **多端**：Windows/Mac 桌面端、Web 端；
- **AI 智能私教**：LLM + 音乐知识，自然语言答疑、个性化学习计划、错误诊断；
- **创新功能**：编曲/改编、AI 自动生成练习曲、合奏模式。

## 风险与依赖（跨阶段）

| 风险 | 影响阶段 | 对策 |
|------|---------|------|
| 乐音识别性能不达标 | Phase 1 | PoC 前置、下沉原生、参数调优 |
| 内容/版权生产慢 | Phase 1–3 | 结构化曲谱工具提效；优先公有领域曲目；UGC 补充 |
| 用户留存难 | Phase 2 | 游戏化与社交前置到 V1 |
| 对手先发优势 | 全程 | 差异化（低价/更好反馈/尤克里里垂直深度）+ 社区 |
