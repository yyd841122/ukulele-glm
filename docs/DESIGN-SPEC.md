# DESIGN-SPEC · 尤克里里 AI 学园 — UI/UX 设计规范

> **版本**：v1.0  ｜  **创建**：2026-06-26
> **设计基调**：**活力年轻 · 渐变风** —— 让学琴有趣、留住年轻人，对标 AI 音乐学园的年轻化调性。
> **适用**：原型（`prototype/index.html`）与未来 Flutter App 的视觉与交互基线。

---

## 1. 设计原则

| 原则 | 说明 |
|------|------|
| **趣味优先** | 游戏化氛围，奖励即时可见，降低练琴枯燥感 |
| **反馈即视** | 任何弹奏动作都有即时、醒目的视觉反馈（颜色/动画） |
| **大而清晰** | 移动端单手操作，关键元素大尺寸、高对比 |
| **专注内容** | 曲谱/和弦图是主角，UI 不喧宾夺主 |
| **暖色治愈** | 橙黄+青绿，温暖、阳光、夏威夷海岛感 |

## 4. 间距与圆角

### 间距（4 倍数体系）
`4 / 8 / 12 / 16 / 20 / 24 / 32 / 40`（单位 px）
- 页面边距：**16px**（两侧）
- 卡片内边距：**16px**
- 元素间纵向间距：**12px**

### 圆角
| 用途 | 圆角 |
|------|------|
| 卡片 | 16px |
| 按钮 | 12px（小）/ 999px（胶囊大按钮） |
| 输入框 | 12px |
| 头像/徽章 | 50%（圆形） |

### 阴影
- 卡片：`0 2px 12px rgba(0,0,0,0.06)`
- 浮起按钮：`0 8px 24px rgba(255,107,44,0.30)`

## 5. 核心组件规范

### 5.1 底部 Tab 导航（5 项）
- 固定底部，高度 56px + 安全区；
- 5 项：🏠 首页 / 🎓 学习 / 🎸 练琴 / 🎼 曲谱 / 👤 我的；
- 选中态：主橙 `#FF8A3D` + 图标填充 + 下方小圆点；未选中：灰色 `#9CA3AF`。

### 5.2 卡片（Card）
- 白底、圆角 16px、浅阴影；
- 标题（H3）+ 副信息（Caption）+ 可选操作。

### 5.3 主按钮
- 渐变背景 `#FF8A3D→#FF6B2C`，白字，胶囊形（圆角 999），高 48px；
- 禁用态：`#E5E7EB` + 灰字。

### 5.4 调音器表盘（核心组件）
- 圆形表盘，刻度 -50 ~ +50 cents；
- 指针：偏左红/居中绿✓/偏右红；
- 中央大字显示音名（如 G）+ cents 数值；
- 下方 4 弦进度（G-C-E-A），当前引导弦高亮。

### 5.5 和弦图
- 4 列（对应 4 弦）× 4~5 品格网格；
- 黑点表示按弦位置，顶部 ×/○ 表示闷音/空弦；
- 圆角 12px，白底，可点击听音。

### 5.6 评分反馈
- 实时：当前音符位置出现 ✓（绿）/✗（红）气泡；
- 报告：环形进度（总分）+ 三维分项条（音准/节奏/完成度）。

## 6. 页面布局结构（5 大主页面 + 详情/弹层）

| 页面 | 布局要点 |
|------|---------|
| 首页 | 顶部渐变问候条 → 打卡入口 → 继续学习卡 → 快捷工具宫格(4) → 推荐曲目横滑 → 热门榜单 |
| 学习 | Tab：必修/选修/Pro → 课程卡片列表（封面+进度条+难度） |
| 练琴 | 工具宫格（调音器⭐最大卡片/节拍器/和弦库/节奏练习/和弦转换） |
| 曲谱 | 顶部搜索+筛选(难度) → 曲谱列表卡（封面/难度星/标签） |
| 我的 | 渐变头部（头像+等级+经验条）→ 数据看板(3格) → 打卡日历 → 成就墙横滑 |
| 曲谱详情 | 顶部曲目信息 → 滚动播放器（和弦+歌词+节奏型）→ 底部跟弹/变速控制 |
| 调音器(独立页) | 顶部弦序选择 → 大表盘 → 4 弦状态 → 开始/停止按钮 |

## 7. 交互与动效

| 场景 | 动效 |
|------|------|
| Tab 切换 | 页面横向淡入 |
| 打卡成功 | 卡片渐变亮起 + 烟花/勋章弹出 |
| ✓ 弹对 | 绿色 ✓ 弹性放大消失 |
| 调音指针 | 平滑跟随音高（CSS transform，去抖） |
| 按钮点击 | 轻微缩放 0.96 |
| 数字滚动 | 评分/经验值 count-up 动画 |

动效时长统一：快 150ms / 标准 250ms / 强调 400ms，缓动 `ease-out`。

---

## 8. 适配

- 设计基准宽：**375px**（iPhone 标准），原型以此宽居中手机框展示；
- 最小支持 360px，最大内容区 480px（平板可居中留白）；
- 安全区底部适配（底部 Tab 留 34px home indicator 空间）。

---

## 9. 图标

- 风格：圆润线性 + 部分填充，与年轻活力调性一致；
- 推荐：原型用 Emoji 快速表达语义；正式 App 用自绘 SVG 图标库（24px 基准）。
